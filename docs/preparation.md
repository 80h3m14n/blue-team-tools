The preparation phase in blue teaming involves establishing foundational security measures before an incident occurs. 

This includes:
- Knowing your network topology & baseline normal network behavior
- Hardening operating systems, patch, set firewall rules, deploy EDR/SIEM
- Enabling early warning indicators
- Developing incident response plans with defined roles and procedures



## Categories of defense mechanisms

Network Defense Essentials (NDE) encompasses a comprehensive set of mechanisms designed to protect network infrastructure from unauthorized access, modification, destruction, or disclosure

### 1. Technical controls

- Network segmentation
- Firewalls
- Intrusion Detection Systems (IDS)
- Intrusion Prevention Systems (IPS)
- Honeypots
- Proxy servers
- Virtual Private Networks (VPNs)
- Security Information and Event Management (SIEM) systems
- User and Entity Behavior Analytics (UBA)
- Anti-malware software.

### 2. Physical

Physical security measures, workplace security, and environmental controls to safeguard the physical infrastructure.


### 3. Administrative

Frameworks, laws, acts, and security policies that guide organizational security practices
